Test Data for acefile
=====================

This repo contains a small corpus of ACE archives intended for testing
[droe/acefile](https://github.com/droe/acefile), both manually by developers,
and automatically by the CI pipeline.  Archives in this repository are
automatically used for pytest based smoke testing.

The contents of the archives in this repo may be licenced under various
permissive licenses.  No proprietary or confidential data.
